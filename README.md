## 1. Create & activate the venv

### macOS/Linux
```
python3 -m venv onboarding_env
source onboarding_env/bin/activate       
```
### Windows
```
onboarding_env\Scripts\activate        
```
## 2. Install packages
```
pip install -r requirements.txt
```

## 3. Run the check
```
python3 onboard.py
```
